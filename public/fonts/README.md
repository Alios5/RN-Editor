# TASA Explorer Font Installation

## Required Files

Please place the TASA Explorer font files in this folder with the following names:

### Regular Weight (400)
- `TASAExplorer-Regular.woff2`
- `TASAExplorer-Regular.woff`

### Medium Weight (500)
- `TASAExplorer-Medium.woff2`
- `TASAExplorer-Medium.woff`

### SemiBold Weight (600)
- `TASAExplorer-SemiBold.woff2`
- `TASAExplorer-SemiBold.woff`

### Bold Weight (700)
- `TASAExplorer-Bold.woff2`
- `TASAExplorer-Bold.woff`

## Note

- `.woff2` is the preferred format (better compression)
- `.woff` is a fallback for older browsers
- The application will use TASA Explorer for all text except titles, which use Audiowide from Google Fonts
